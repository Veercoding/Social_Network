📊 Complexity Analysis

Time and Space complexity of all data structures and algorithms used

Data Structure 1 — Hash Table

Operation Time Complexity Space Complexity Used For
Insert user O(1) average O(n) Adding new user profile
Lookup user O(1) average O(1) Login, profile view
Delete user O(1) average O(1) Remove account
Worst case O(n) O(n) Hash collision chain

Data Structure 2 — Graph (Adjacency List)

Operation Time Complexity Space Complexity Used For
Add node (user) O(1) O(1) New user joins
Add edge (friendship) O(1) O(1) Friend request accepted
Remove edge O(E) O(1) Unfriend
Storage total — O(V + E) Entire network

Algorithm 1 — Breadth-First Search (BFS)

Case Time Complexity Space Complexity Used For
All cases O(V + E) O(V) Friend suggestions
Six degrees O(V + E) O(V) Shortest path finding

Algorithm 2 — Insertion Sort

Case Time Complexity Space Complexity Used For
Best case O(n) O(1) Already sorted list
Average case O(n²) O(1) Friend suggestions ranking
Worst case O(n²) O(1) Leaderboard ranking
